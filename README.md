# MangaStore Chile - Ecommerce Frontend MVP 🇨🇱

Primera versión (MVP) de una tienda en línea de mangas y cómics, desarrollada para aplicar conceptos de HTML5 semántico, diseño responsivo con Bootstrap y manipulación dinámica del DOM con JavaScript.

## 🚀 Funcionalidades
- **Catálogo Dinámico:** Los productos se renderizan desde un arreglo de objetos en JS.
- **Carrito de Compras:** Contador en tiempo real en la navbar.
- **Persistencia:** Uso de `localStorage` para que no se pierdan los datos al recargar.
- **Vista de Detalle:** Modal interactivo con información expandida de cada producto.
- **Responsive Design:** Adaptado para móviles (Mobile-first) y escritorio.

## 🛠️ Tecnologías utilizadas
- **HTML5:** Estructura semántica avanzada.
- **Bootstrap 5:** Sistema de grillas, componentes (Cards, Modales, Nav) y utilidades.
- **JavaScript (Vanilla):** Lógica de negocio, eventos y manejo de arreglos.
- **Git/GitHub:** Control de versiones.
